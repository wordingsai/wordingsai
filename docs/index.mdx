## 🚀 Getting Started

To run the development environment locally:

### Prerequisites

- Node.js 18+
- PNPM (for package management)

### Installation

```bash
# Install dependencies
pnpm install
```

### Development

```bash
# Start the Next.js development server
pnpm dev
```

### Environment Variables

Ensure you have a `.env` file in the root with the required API keys (OpenRouter, Supabase, Neon).
